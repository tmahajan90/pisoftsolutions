## ğŸ“ Step-by-Step Setup

### 1. DNS Configuration

Configure your domain's DNS records to point to your server:

```bash
# A Record
Type: A
Name: @
Value: YOUR_SERVER_IP

# A Record for www subdomain
Type: A
Name: www
Value: YOUR_SERVER_IP

# CNAME Record (alternative)
Type: CNAME
Name: www
Value: pisoftsolutions.in
```

### 2. Server Preparation

